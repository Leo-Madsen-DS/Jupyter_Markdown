Las variables complejas no se pueden convertir en nmeros flotantes o enteros. Necesitamos usar z.real o z.imag para extraer la parte del nmero complejo que queremos:

Si omitimos uno (o ambos) de start o stop desde [start:stop], el valor predeterminado es el principio y el final de la cadena, respectivamente:

In [ ]:

In [ ]:

Tambin podemos definir el tamao del paso usando la sintaxis [start:end:step]

In [ ]:
s[::1]
In [ ]:
s[::2]

Booleanos¶
  • Los operadores booleanos se escriben como las palabras and, not, or.
Operador Uso Descripcin
and a and b Devuelve True si ambos operandos son True
or a or b Devuelve True si alguno de los operandos es True
not not a Devuelve True si alguno de los operandos False

b = False
print(a and b) #False

True
